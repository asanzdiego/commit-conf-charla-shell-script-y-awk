# Shell Script




## Hola mundo

- Un shell script es un fichero de texto con comandos Linux.

~~~
#! /bin/bash

echo "Hola mundo!"
~~~

## Permisos

- Antes de ejecutar hay que darle permisos.

~~~
$ chmod +x 01-hola-mundo.sh
~~~

## Ejecución

- Si está en el PATH, el nombre directamente, sino ./nombre

~~~
$ ./01-hola-mundo.sh
~~~
